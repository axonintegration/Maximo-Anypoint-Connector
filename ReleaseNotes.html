Release Notes for version 1.0.0 of the Maximo connector. These Release Notes accompany the <Connector_Guide_URL> guide.

Compatibility

The Maximo connector is compatible with:

Application/Service

Version

Maximo

7 and above

Mule Runtime

3.6.x

Maximo Connector

1.0.0

Migrating from older versions of the connector:

Not applicable as this is the first public release

Features

  1. Object Structure based web service - Execute a deployed web service that is based on a Maximo object structure

  2. Query and Update operations - Execute a Maximo query, add, delete, change or addChange on a Maximo entry (or batch of).

  3. Authentication and Security - Supports native authentication through a Maximo username and password authenticated account. Supports HTTPS through the setting of certificates in the Mule truststore and setting of an HTTPS Maximo web app URL.

Fixed in this Release

Not applicable as this is the first public release

Known Issues

  • No support for customized web services of web services exposing Enterprise Services - Contact Computime Software if support is required (www.computime.com.mt)

  • No support of WebSphere MQ integration - Contact Computime Software if support is required (www.computimesoftware.com)
